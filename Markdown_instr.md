## Списки

Чтобы добавить ненумерованные списки, необходимо выделить звездочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо выделить пункты просто пронумеровать. Например, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
